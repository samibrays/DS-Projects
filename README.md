# Supervised Machine Learning with Insurance Dataset

This project demonstrates end-to-end supervised machine learning on an insurance dataset. It includes regression and classification models, hyperparameter tuning, and model evaluation.

## Features

- Exploratory Data Analysis (EDA)  
- Data cleaning & feature engineering  
- Multiple supervised ML algorithms:
  - Linear Regression, Ridge, Lasso
  - Logistic Regression
  - Decision Tree, Random Forest
  - SVM, KNN
  - Naive Bayes  
- Model evaluation and comparison  
- Insights for insurance scenarios
